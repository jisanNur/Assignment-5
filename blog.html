What is the Document Object Model (DOM)?

DOM (Document Object Model) is a hierarchical representation of HTML documents. It defines the structure and properties of a webpage's elements, enabling JavaScript to dynamically access, manipulate, and update content, enhancing interactivity and functionality.

How do you select an element from the DOM?

1. getElementById()

2. getElementBysclassName()

3. getElementByTagName()

4. querySelector()

5. querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you manipulate an element's attributes and styles using the DOM?

To change other attributes like id, href or src use the setAttribute() method. The setAttribute() method takes two arguments: the name of the attribute to set. Attribute value to set.
